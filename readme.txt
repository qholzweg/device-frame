Frames your image with a mobile device screen

== Description ==

Allows you to choose a device from a list of Apple devices and upload a screenshow to be shown on a screen of the chosen device

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/device-frame` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
